# alex-cms-buildout

The custom CMS for **alexmchong.ca** — a single-author personal site (designer / writer / coach). PHP + MySQL on DreamHost shared hosting.

## Where to start

- **`CLAUDE.md`** — orientation guide. Read first.
- **`docs/BUILD-PLAN.md`** — the 17-phase execution plan. §3 has the phase index; find the first unchecked box for the current phase.

## What this repo contains

| Path | What it is |
|---|---|
| `docs/CMS-STRUCTURE.md` | Canonical system spec — schema, views, render rules. |
| `docs/ENGINEERING.md` | Code conventions. |
| `docs/LEGACY-ROUTES.md` | URLs from the old site that must keep resolving. |
| `docs/design-mockups/` | CMS admin panel mockup. |
| `site/_templates/` | Public-site rendering templates (HTML + CSS). |
| `site/_design-system/` | Brand design system (tokens, components, showcase). |
| `site/_pages/` | Standalone marketing pages (outside the CMS). |
| `docs/onboarding/` | Phase 0 setup lessons. |

## Status

Phase 0 in progress. No application code exists yet — Phase 1 ships the static site.
