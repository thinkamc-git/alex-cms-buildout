Eyebrow / section-header options

Four ways to render the recurring "kicker + heading" pattern used throughout the thinking-system piece (e.g. "The problem" → "Projects fail before they begin"). V0 is what's live now (no eyebrow — heading only). A/B/C reuse an existing design-system eyebrow class as-is. D is a proposed dedicated variant, tuned specifically for a repeating in-body section header rather than a listing/hero context.

All samples use the real Barlow / Barlow Condensed fonts and this content's actual category colour (Case Study teal) where a variant ties to --c-current.
